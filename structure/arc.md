# Story Structure & Arc Map

> The skeleton scenes hang on. Two frameworks below — use **Hero's Journey** for the overall
> series spine and **kishōtenketsu** for individual chapters/arcs (it's the manga-native shape
> and works without forcing conflict into every beat). Map your beats to scene IDs so you can
> see gaps before you draft into them.

---

## Framework A — Hero's Journey (12 beats, Vogler)
Best for the **series-level** arc and the protagonist's transformation.

| # | Beat | Function | Scene(s) | Status |
|---|------|----------|----------|--------|
| 1 | Ordinary World | establish hero + the lie they believe |  |  |
| 2 | Call to Adventure | the inciting disruption |  |  |
| 3 | Refusal of the Call | fear, what's at stake |  |  |
| 4 | Meeting the Mentor | guidance / the tool |  |  |
| 5 | Crossing the Threshold | commitment, no going back |  |  |
| 6 | Tests, Allies, Enemies | the new world's rules in action |  |  |
| 7 | Approach to the Inmost Cave | prep for the big ordeal |  |  |
| 8 | The Ordeal | the midpoint death/rebirth |  |  |
| 9 | Reward | they seize the prize |  |  |
| 10 | The Road Back | consequences chase them |  |  |
| 11 | Resurrection | the climax; the lie dies, the truth wins |  |  |
| 12 | Return with the Elixir | changed hero, new normal |  |  |

> Theme check: beat 1 plants the lie, beat 11 proves the moral. If they don't connect, the
> story has no spine.

## Framework B — Kishōtenketsu (4 acts)
Best for **chapters / self-contained arcs**, especially manga. No villain required — it turns
on a *twist*, not a conflict, which suits slice-of-life and mystery beats too.

| Act | Name | Function | Scene(s) |
|-----|------|----------|----------|
| 起 Ki | Introduction | set the scene, who/where/normal |  |
| 承 Shō | Development | deepen it, build texture (no big change yet) |  |
| 転 Ten | Twist | the unexpected turn — the heart of the chapter |  |
| 結 Ketsu | Conclusion | reconcile the twist with the setup; new meaning |  |
