# Power System — Resonance

## Sanderson compliance

**0th Law — keep it awesome.** Hearing what an object remembers — and being able to give it a
new command in someone else's stolen voice — is the wonder. Rules must never sand that off.

**1st Law — understood before it solves.**
- Hard (may resolve conflict): PWR-001 (read), PWR-002 (command). Both are shown, with costs, in
  the first half of the arc, so the climax can legitimately turn on them.
- Soft (atmosphere only): the "deep memory" of drowned places — felt, never used to win.

**2nd Law — costs, limits, weaknesses > powers.** Filled per ability below.

**3rd Law — depth over breadth.** New problems are solved by *combining* read + command on a
single object, never by inventing a new power. The S01-006 climax is exactly this.

## The system in one paragraph
> **Resonance:** a Resonator can press intent into matter by touch. *Reading* replays the last
> intent an object holds; *commanding* writes a new one it will obey once. Every command burns a
> measured length of the user's life-thread — spend too much and you age, then you stop.

## Source & access
- Where it comes from: born to a few; the undercity calls them Resonators.
- Who can use it: a rare minority; the Wardenate registers and leashes them.
- How one grows it: discipline (a tuner's training), not raw effort — this gates power-creep.
- How it's lost or sealed: deafness severs *commanding others*; you can still read and command objects.

## Global laws (apply to all users)
| # | Law | Consequence if broken in-story |
|---|-----|--------------------------------|
| 1 | A command costs life-thread, always | over-spending ages the user; a long command can kill |
| 2 | The deaf cannot be commanded | any "command Kai to be quiet" beat is illegal (she's deaf) |
| 3 | An object obeys a command once, then forgets | no infinite-loop exploits |

## Abilities

### PWR-001 — Resonance (read)
- Effect: replays the last intent/voice pressed into a touched object.
- **Cost:** a wave of vertigo; longer reads bring borrowed grief.
- **Limit:** only the *most recent* strong intent; deep/old memory is unreadable.
- **Weakness:** water muffles it; a fully drowned object goes quiet.
- Tells / visual signature: object glows faint, a low hum, the reader's eyes unfocus.
- Tier / rarity: common among Resonators.
- Known users: CHR-001, CHR-002
- First shown: S01-001

### PWR-002 — Resonance (command)
- Effect: writes one new intent into an object, which it obeys once.
- **Cost:** burns a measured length of life-thread — visible as a streak of hair going white.
- **Limit:** one object, one command, one time; cannot command a living person who is deaf (Law 2).
- **Weakness:** the cost is fixed by the command's force — a big command can be fatal.
- Tells / visual signature: a white streak blooms in the user's hair; the object flares.
- Tier / rarity: rarer; requires training to survive.
- Known users: CHR-001, CHR-002, CHR-003
- First shown: S01-004

### PWR-003 — Tuning (dampen)
- Effect: deadens an object's resonance so it can't be read or commanded.
- **Cost:** exhaustion; a tuner can only hold a few forks "open" at once.
- **Limit:** dampens, never erases — lift the tuning and the memory returns.
- **Weakness:** useless against a command already in motion.
- Tells / visual signature: a struck brass fork; the hum flattens to silence.
- Tier / rarity: a tuner's specialty.
- Known users: CHR-002
- First shown: S01-003
